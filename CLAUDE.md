# 노노그램 미니게임 프로젝트

## 프로젝트 개요
외부 API/라이브러리 없이 순수 HTML/CSS/JS로 만든 웹 기반 노노그램(Picross) 퍼즐 게임.
알고리즘으로 무한히 새 퍼즐을 생성하며, 유일해를 보장한다.

## 파일 구조
```
minigame/
├── index.html       — UI 구조 (한국어 인터페이스)
├── style.css        — 디자인 (라벤더 그라디언트 테마)
├── server.py        — 로컬 개발 서버 (port 3001)
└── js/
    ├── solver.js    — 라인 솔버 + 유일해 검증
    ├── generator.js — 랜덤 퍼즐 무한 생성
    ├── sound.js     — Web Audio API 사운드 (외부 파일 없음)
    └── game.js      — 게임 로직 전체
```

## 개발 서버 실행
```bash
python3 server.py
# → http://localhost:3001 에서 확인
```

## 현재 구현된 기능
- 난이도 3단계: 쉬움(5×5), 보통(10×10), 어려움(15×15)
- 무한 퍼즐 생성 (라인 솔버 기반, 유일해 보장)
- 조작: 왼클릭=채우기, 오른클릭=× 표시, 드래그=연속 조작
- 시작 오버레이 — 버튼을 눌러야 타이머 시작
- 진행 저장/복원 — localStorage, 새로고침 후 "이어하기" 가능
- 오류 표시 토글, 힌트 (랜덤 셀 공개)
- Web Audio API 사운드 8종 (셀 채우기/지우기/X 표시/실수/행열 완성/퍼즐 완성/힌트)
- 🔊 뮤트 버튼 (stats 바 우측)
- 완성 배너 + 다음 퍼즐 버튼
- 모바일 터치 지원
